# Responsive Image Gallery

## Description
This project is a responsive image gallery built using HTML, CSS, and JavaScript.
It includes hover effects, lightbox view, image navigation, and category filtering.

## Features
- Responsive image grid layout
- Hover effects and smooth transitions
- Lightbox image preview
- Next and Previous navigation
- Image category filters (Nature, City)

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in any web browser
3. Click images to view them in lightbox mode

## Output
- Displays images in a grid layout
- Opens images in a full-screen lightbox
- Works on mobile, tablet, and desktop
